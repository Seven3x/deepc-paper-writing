# Paper Subrepository

This repository contains paper-oriented materials for the DeePC quadcopter project.

## Contents

- `title_candidates.md`: conservative title options for the active paper line
- `abstract_skeleton.md`: abstract boundary and claim skeleton
- `contributions.md`: evidence-backed contribution list
- `outline.md`: section-by-section paper structure
- `introduction_skeleton.md`: introduction draft skeleton
- `method_skeleton.md`: method draft skeleton
- `experiments_skeleton.md`: experiment protocol skeleton
- `results_skeleton.md`: result organization skeleton
- `discussion_skeleton.md`: discussion and limitations skeleton
- `manuscript/`: main writeup drafts
